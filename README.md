Funcionalidades e Estrutura
Classe Viatura: Define a estrutura de um veículo, contendo atributos como marca, modelo, matrícula, tipo de combustível, quilometragem e preço. Inclui métodos para aceder e modificar cada um desses dados.

Classe Dados: Atua como o "banco de dados" do sistema. Esta classe armazena uma lista de viaturas e possui métodos para:

Adicionar novas viaturas, com validação para impedir matrículas duplicadas.

Remover viaturas usando a matrícula como identificador único.

Listar todas as viaturas registradas.

Filtrar e mostrar viaturas de uma marca específica.

Calcular e apresentar o valor total de todas as viaturas registradas.

Interface e Testes: O projeto inclui uma classe principal que utiliza caixas de diálogo para interagir com o utilizador e testar todas as funcionalidades das classes anteriores.
